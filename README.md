Programming Assignments

A well-organized collection of programming assignments across Operating Systems, Java, and Database Management Systems (DBMS), focusing on core concepts, problem-solving, and practical implementations.

---

Subjects Covered

Operating Systems (C)

* Process creation and management (`fork`, `exec`, etc.)
* Multithreading using pthreads
* Synchronization (mutex, locks)
* Process scheduling concepts

 Java

* Object-Oriented Programming (OOP)
* Collections Framework
* Exception Handling
* Basic application development

Database Management Systems (DBMS)

* SQL queries (DDL, DML, DCL)
* Joins and subqueries
* Normalization
* Database design fundamentals

---

 Tech Stack

* Languages: C, Java, SQL
* Concepts: OS programming,DSA, OOP, Database Design
* Tools: GCC, MySQL (or any SQL environment)
